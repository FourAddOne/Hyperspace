# OSS 生命周期规则配置指南

为了实现聊天文件3天后自动删除的功能，需要在阿里云OSS控制台配置生命周期规则。

## 配置步骤

1. 登录阿里云控制台
2. 进入对象存储OSS服务
3. 找到并进入名为 `fourandone-hyperspace` 的Bucket
4. 在左侧菜单中选择"生命周期规则"
5. 点击"创建规则"
6. 配置以下规则：

### 规则1：删除聊天文件
- 规则名称：DeleteChatFiles
- 前缀：files/
- 状态：启用
- 策略：
  - 操作：删除Object
  - 过期时间：3天

### 规则2：删除聊天图片
- 规则名称：DeleteChatImages
- 前缀：messages/
- 状态：启用
- 策略：
  - 操作：删除Object
  - 过期时间：3天

## 注意事项

1. 生命周期规则每天执行一次，通常在UTC 00:00-01:00之间
2. 文件可能在过期时间后的24小时内被删除
3. 一旦配置，规则将自动应用于匹配前缀的新文件
